Quel est notre objectif?

Notre objectif est de fournir un outil en ligne aux étudiants sans expérience professionnelle ou aux équipes de projet pour développer des logiciels via une interface graphique conviviale avec des fonctions prédéfinies et des paramètres adaptables. Une fois la séquence d'exécution de la tâche décrite, vous pouvez facilement exporter le code source de votre programme.
